# Global Chat - Node.js Version

A real-time global chat application using Socket.io

## Setup & Run

### 1. Install Node.js
Download from https://nodejs.org/ (v14 or higher)

### 2. Install Dependencies
```bash
cd chat-app
npm install
```

### 3. Start the Server
```bash
npm start
```

The server will run on `http://localhost:3000`

### 4. Open in Browser
Open http://localhost:3000 in your browser

Multiple people can now open the same URL and chat in real-time!

## Features
- ✅ Real-time messaging with Socket.io
- ✅ Global chat - everyone sees all messages
- ✅ Username support
- ✅ Message timestamps
- ✅ Auto-scrolling
- ✅ Beautiful UI
- ✅ Stores last 500 messages

## Deploy to Cloud (Optional)
To make it accessible to anyone online:

### Heroku
```bash
heroku create your-app-name
git push heroku main
```

### Glitch.com
1. Go to https://glitch.com
2. New Project → Import from GitHub
3. Paste the repo URL

### Replit
1. Go to https://replit.com
2. Import from GitHub
3. Click "Run"

That's it! Share the URL with friends and start chatting globally!
